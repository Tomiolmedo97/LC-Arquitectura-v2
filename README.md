# LC Arquitectura — Sitio web

Sitio institucional de LC Arquitectura (estudio de arquitectura — retail/comercial, residencial premium y hotelería/corporativo).

## Estructura

- `index.html` — página única con secciones Inicio, Nosotros, Servicios y Contacto.
- `assets/` — logo, isotipo, favicon y bocetos de fondo.

Responsive, con menú móvil, animaciones de aparición al hacer scroll y tipografías Montserrat / Cormorant Garamond desde Google Fonts.

## Publicar con GitHub Pages

1. Subir `index.html` y la carpeta `assets/` a la raíz del repositorio.
2. En el repo: **Settings → Pages → Source: Deploy from a branch → Branch: main / (root) → Save**.
3. En unos minutos el sitio queda disponible en `https://<usuario>.github.io/<repo>/`.
